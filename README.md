# BARPO

This repository contains a reference implementation for Bilevel Adversarially Robust Policy Optimization (BARPO). See our paper ["On the Tension Between Optimality and Adversarial Robustness in Policy Optimization"](https://arxiv.org/abs/2512.01228) for more details. This paper has been accepted by [ICLR 2026](https://openreview.net/forum?id=ion4VYJWvo).

Since the author is busy dealing with graduation, the detailed code will be released later.
